Internship Task 1: Web Scraping
Project Title

Web Scraping Quotes and Authors using BeautifulSoup

Objective

The objective of this task is to extract quotes and their corresponding authors from a public website using Python web scraping techniques.

Tools Used

Python

Requests library

BeautifulSoup (bs4)

Steps Followed

Imported required libraries

Identified a public website for scraping

Sent HTTP request to the website

Parsed HTML content using BeautifulSoup

Extracted quotes and authors from the webpage

Displayed the scraped data in a readable format

Website Used

https://quotes.toscrape.com/

Output

The program successfully extracts and displays:

Quotes

Corresponding authors

Conclusion

This task demonstrates the basic concepts of web scraping using Python. It helps in understanding how to collect structured data from public websites, which is useful for data analysis and real-world applications.
