Overview

This internship focuses on developing practical data analytics skills through hands-on tasks such as web scraping and exploratory data analysis. The tasks involve collecting data from public websites and analyzing datasets to identify trends, patterns, and data quality issues using Python-based tools.

Task 1: Web Scraping

Objective:
To extract quotes and authors from a public website using Python.

Tools Used:
Python, Requests, BeautifulSoup (bs4)

Steps Followed:

Sent HTTP request

Parsed HTML

Extracted quotes and authors

Outcome:
Quotes and authors were successfully scraped and displayed.

Task 2: Exploratory Data Analysis (EDA)

Objective:
To understand the dataset and identify patterns before detailed analysis.

Tools Used:
Python, Pandas, Matplotlib

Steps Followed:

Explored dataset structure

Identified trends using visualizations

Tested hypothesis

Checked data issues

Outcome:
Key trends, correlations, and data issues were identified.
